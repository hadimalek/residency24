# Site images (`/public/images`)

Files committed here are published with every build and served at
`https://<domain>/images/<filename>`. They survive deploys (they are part of
the git repo), unlike anything generated under `.next/`.

## Credibility photos used by the About & Contact pages

Commit your real photos here with **these exact names** (JPG/PNG/WebP).
Until a file is added, the page shows a branded Navy placeholder instead of
a broken image — so you can add them at any time.

| File | Where it appears | Suggested shot |
|------|------------------|----------------|
| `office-dubai.jpg` | Contact page — office banner | Dubai office (exterior or interior), wide/landscape |
| `about-1.jpg` | About page — gallery | Office / workspace |
| `about-2.jpg` | About page — gallery | Team at work / meeting |
| `about-3.jpg` | About page — gallery | Reception / signage / city context |

Recommended: landscape ~4:3 or 16:9, ≥ 1200px wide, optimized (< ~400 KB each).

The About "Get to Know Us" video is embedded from YouTube — no file needed here.
